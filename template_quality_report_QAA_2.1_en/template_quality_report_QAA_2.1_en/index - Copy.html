Ins/Unins(tallation)
•  Medium
Although installation and uninstallation are functional, several issues were observed during testing. QATC-910921 - The installer lacks an option to run as an administrator. QATC-910924 - Full installation on a user-mapped drive doesn't consume all space, leaving some on the default drive. Despite these concerns, the core functionality remains intact, ensuring basic usability.
 

Main window & File menu
•  Medium
The quality of module is estimated as Medium. The key issues observed are as follows: QATC-911296 - Selecting more than one picture allows users to edit only one image at a time instead of editing all selected pictures simultaneously. QATC-911293 - Refreshing the main window forgets the changes made to images. Despite these defects, basic functionality remains operational, ensuring usability of the Main Window and File Menu module.
 

Optimize
•  Low
One of the most important modules of the application has significant functionalities not working. QATC-911281 - The auto-optimize button does not function. QATC-911282 - The red-eye reduction button fails to work. QATC-911284 - The sharpness, brightness and contrast sliders are non-functional. With two non-functional buttons and 3 non-functional sliders and other defects, the Optimize module has significant limitations, affecting its overall usability.
 

Options menu
•  Medium
The quality of this section can be estimated as Medium. QATC-911432 - Changing the application language in the Options Menu to German (Deutsch) does not update the user interface to German language. QATC-911654 - Parameter dimension fields do not accept non-integer values. Despite these defects, the Options Menu retains basic functionality, ensuring usability for most tasks.
 

Help menu
•  Low
The quality of this section can be estimated as Low. QATC-911298 - Help button link never works, since it contains a broken link to a Help web-page (Leads to 404 Not found).
 

Create gallery
•  Low
The key issues observed are as follows: It is impossible to understand what the Auto-optimize button does, making it unclear to users. QATC-911710 - The create gallery feature does not function when the application is installed on the C:/ drive, significantly impacting usability. QATC-911573 - Instead of a ZIP file, a folder is created. Despite these defects, the core functionality of the Create Gallery module remains operational if the user installs Kimba application on D:/ drive.
 

GUI
•  Medium
This module doesn't contain any serious defects, but is crucial for user interface and experience.
